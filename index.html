Lichen

Lichen includes a Network Viewer, Rectangular Heatmap and Visualization Template implemented in actionscript and javascript using the google visualization API.

Its is designed to provide a set of browser independent visualizations that can easily be incorportated as figures and user interface elements in scientific data bases and other web applications.

It is primarily the work of a group of software engineers at the Institute for Systems Biology in Seattle, WA but is open to other collaborators. Please feel encouraged to branch our code and contact us through github.

In Use

A Lichen visualization created with R.Lichen is on the front page of http://beta.gbmbase.org.

The demo pages from a 5/27/2009 Research in Progress at the Institute for Systems Biology are now included in the github pages here.

The heatmap is used in the meta analysis tool on t1dbase.org. You can see an example here.

An early version of the network viewer is used on the gene overview pages on t1dbase.org, it is in a collapsed tab in the tool section. INS and STAT5B are good examples.
